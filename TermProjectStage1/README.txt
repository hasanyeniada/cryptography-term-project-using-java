(Note:Documents should be in the same path with the jar file)

There are also ExtendedEuclideanTest class which you can test the results of
gcd, multiplicative inverse and relatively prime methods of the ExtendedEuclidean class.
You can extract the project from jar and test them, because the main function of the jar
only includes the implementation of the given scenario.

To execute the given jar, 
1) First option is to double click, but this does not show the console results.
2) Open command line, go to path which includes the jar file and 
the test documents (Note:they should be in the same path), enter the following command:
java -jar CENG471_TermProject_Stage1.jar

After execution of the jar, you can compare the original documents with the decrypted documents.
